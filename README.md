# Ecommerce Website

A simple and efficient ecommerce web application built with a clean frontend and a fully functional backend. It covers all essential features required for a basic online shopping experience, including product display, cart handling, user actions and database support.

## Features

* Modern and responsive UI
* Product listing with images and details
* Add to cart and remove from cart
* Dynamic cart updates
* Server side routing for all pages
* Clean folder structure
* Secure backend operations
* SQL database integration
* Fast and smooth user experience

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node js
* Express js

### Database

* SQL

## How to Run

1. Clone the repository
2. Install dependencies

```
npm install
```

3. Set up your SQL database
4. Update database credentials in config
5. Start the server

```
npm start
```

6. Visit in browser

```
http://localhost:3000
```
